# NOIR Coffee Landing Page

Landing page desenvolvida durante meus estudos de HTML e CSS no Responsive Web Design Certification do freeCodeCamp.

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Responsive Design

## Objetivos do projeto

- Criar uma landing page moderna
- Praticar estrutura semântica em HTML
- Aplicar responsividade com media queries
- Desenvolver uma identidade visual consistente

## Funcionalidades

- Navegação fixa
- Layout responsivo
- Seção de recursos
- Vídeo incorporado
- Planos de produto
- Formulário de captura de e-mail
